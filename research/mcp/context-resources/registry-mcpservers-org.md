# mcpservers.org

## Identification
- url: https://mcpservers.org/
- type: registry
- author: community
- last-updated: active
- authority level: community

## Scope
Community-curated MCP server marketplace. Organized by function: Search, Web Scraping, Communication, Productivity, Development, Database, Cloud Service, File System, Cloud Storage, Version Control, Other. Marks "Official" servers with a star. Has Featured and All views, sortable by newest.

## Takeaway summary
Another community directory — not Anthropic-run despite the `mcpservers.org` domain. Editorial curation alongside community submissions; supports both free and sponsored listings. Good as a category-browsable discovery surface when you know the functional domain but not specific server names. Not authoritative for schema or protocol details — for those go to modelcontextprotocol.io or the official registry.

## Use for
- Browse MCP servers by function (search, database, file system, etc.)
- Discover mainstream vendor integrations (Google, Microsoft, GitHub) alongside emerging tools
- Secondary discovery surface when the official registry doesn't surface what you need

## Relationship to other resources
Peer of Glama, mcp.so, PulseMCP, Smithery. Overlaps heavily in listings.

## Quality notes
Submission process isn't explicitly documented on the site. The "Official" star is editorial judgment, not a formal protocol designation.
